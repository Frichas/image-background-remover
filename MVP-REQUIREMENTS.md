# Background Remover - MVP 需求文档

## 📋 项目概述

**产品名称**: Background Remover  
**目标关键词**: image background remover  
**核心功能**: 在线 AI 抠图工具，一键移除图片背景  
**技术栈**: Next.js + Cloudflare Pages + Remove.bg API  
**MVP 目标**: 2周内上线可用产品，验证市场需求

---

## 🎯 产品定位

### 目标用户
- 电商卖家（需要处理商品图）
- 自媒体创作者（制作封面/配图）
- 设计师（快速抠图需求）
- 普通用户（证件照、头像处理）

### 差异化策略
1. **极简体验** - 无需注册，打开即用
2. **快速响应** - Cloudflare 全球 CDN 加速
3. **隐私优先** - 图片不存储，处理完即删
4. **免费额度** - 每日 3 次免费，低门槛试用

---

## ✅ MVP 核心功能（必须有）

### 1. 图片上传
- 支持拖拽上传 / 点击选择
- 支持格式：JPG, PNG, WEBP
- 单张限制：10MB
- 实时预览原图

### 2. 背景移除
- 调用 Remove.bg API 处理
- 显示处理进度（loading 状态）
- 错误提示（API 失败、格式不支持等）

### 3. 结果展示
- 左右对比视图（原图 vs 抠图结果）
- 透明背景预览（棋盘格背景）
- 下载按钮（PNG 格式）

### 4. 免费额度限制
- 每日 3 次免费（基于 IP 限制）
- 超出后显示升级提示
- 简单计数器展示剩余次数

## 🔧 技术实现

### 前端
- **框架**: Next.js 14 (App Router)
- **样式**: TailwindCSS
- **上传**: react-dropzone
- **对比**: 自定义左右分屏组件

### 后端
- **部署**: Cloudflare Pages
- **API**: Cloudflare Functions
- **抠图**: Remove.bg API
- **限流**: Cloudflare KV (存储 IP 计数)

### 数据流
```
用户上传图片 
  → 前端转 Base64 
  → POST /api/remove-bg 
  → Cloudflare Function 调用 Remove.bg 
  → 返回处理后的图片 Base64 
  → 前端展示 + 下载
```

## 🔐 风险与应对

### 风险 1: Remove.bg API 成本过高
**应对**: 
- 严格限制免费额度
- 后续接入更便宜的 Clipdrop API
- 考虑自部署 BiRefNet 模型

### 风险 2: 滥用（恶意刷量）
**应对**:
- IP 限流（Cloudflare KV）
- 添加简单的 CAPTCHA
- 监控异常流量

### 风险 3: SEO 竞争激烈
**应对**:
- 专注长尾词（"background remover for product photos"）
- 快速迭代，增加差异化功能
- 社交媒体推广（Reddit, ProductHunt）
